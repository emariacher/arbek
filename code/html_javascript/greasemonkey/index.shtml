Eric Mariacher: GreaseMonkey javascripts for online business tools (LinkedIn, OpenBC, Viaduc...)

Pour utiliser ces scripts il faut:
  1. D'abord installer firefox et ... utilisez le à la place d'Internet Explorer
  2. Puis installer l'extension greasemonkey pour FireFox
  3. Et enfin clicquer sur les scripts du tableau ci-dessous
 To use these scripts:
  1. Install firefox and ... use it instead of Internet Explorer
  2. Install FireFox extension greasemonkey
  3. Click on user's scripts in table below
Greasemonkey javascriptDescriptionUser's guide
searchloz.user.js

Search Online Business Networking sites


Aggregate Searches sur LinkedIn, OpenBC and ZoomInfo


@include http://search.live.com/*
@namespace http://mariacr.multimania.com/scripts 		Type your search in search.live.com/macros/biznet1/biznet1 search page. A new tab will be opened with search results in:
  • LinkedIn
  • OpenBC
  • Zoominfo
searchobn.user.js

Recherche dans les sites de réseautage en ligne


Faites des recherches en parallèle sur LinkedIn, OpenBC, Viaduc et ZoomInfo


@include http://search.live.com/*
@namespace http://mariacr.multimania.com/scripts 		Tapez votre recherche sur search.live.com/macros/biznet1/biznet1. Les résultats seront affichés dans un nouvel onglet:
  • LinkedIn
  • OpenBC
  • Viaduc
  • Zoominfo
contactsviaduc.user.js

Contacts Viaduc


Affiche sur une seule page tous mes contacts Viaduc


@include http://www.viaduc.com/contacts/contactsdirects/*
@namespace http://mariacr.multimania.com/scripts
  1. Allez à http://www.viaduc.com/contacts/contactsdirects/.
  2. Un bouton vert apparaitra en surimpression en haut à droite de la page Viaduc. Cliquez le.
  3. Vous verrez les numéros de page de vos contacts défiler dans le bouton vert
  4. Quand le compte se stabilise, Cliquez une 2ème fois sur le bouton. (En général il doit se stabiliser au numéro de votre dernière page de contacts.)
  5. Cela devrait déclencher l'ouverture d'un ou plusieurs onglets contenant un tableau simplifié de vos contacts.
  6. Vous pouvez faire un copier/coller de l'ensemble de la page dans Excel.
